# 👩‍🏫 教師專用操作指南

## 🚀 快速開始（5分鐘設定）

### 第一步：登入管理後台
1. 前往：`http://work.100thy.com/admin_login.php`
2. 輸入帳號：`admin`
3. 輸入密碼：`admin123456`
4. 點擊「登入」

### 第二步：修改密碼（重要！）
1. 登入後點擊「系統設定」
2. 在「安全設定」區域修改管理員密碼
3. 建議使用包含大小寫字母、數字的強密碼

### 第三步：創建教室
1. 點擊「教室管理」
2. 填寫教室名稱（例如：高一甲班）
3. 設定分享代碼（例如：CLASS2025A）
4. 點擊「創建教室」

### 第四步：設定觀摩模式
1. 點擊「系統設定」
2. 選擇觀摩模式：
   - **手動審核**（推薦）：需要您審核才顯示
   - **自動公開**：學生上傳後立即顯示
   - **完全關閉**：不顯示任何作業

### 第五步：分享給學生
將以下連結提供給學生：
```
http://work.100thy.com/upload.php?code=您的教室代碼
```

## 📚 日常管理流程

### 📝 作業管理
1. **查看新作業**
   - 點擊「作業管理」
   - 查看「待評分」的作業

2. **評分作業**
   - 點擊作業的「評分」按鈕
   - 輸入分數（0-100）
   - 添加評語（可選）
   - 點擊「儲存評分」

3. **審核觀摩**
   - 點擊作業的「觀摩」按鈕
   - 選擇「通過觀摩」或「拒絕觀摩」
   - 優秀作業可設為「精選」

### 🏫 教室管理
1. **查看教室統計**
   - 在教室管理頁面查看各教室的作業數量
   - 複製分享代碼給學生

2. **管理教室**
   - 編輯教室名稱和描述
   - 啟用/停用教室
   - 刪除空教室（無作業的教室）

### ⚙️ 系統設定
1. **觀摩模式**
   - 根據教學需要調整觀摩模式
   - 建議課程初期使用「手動審核」

2. **評分設定**
   - 調整最高分數
   - 設定分數是否對學生公開
   - 啟用/停用評分備註功能

## ❓ 常見問題解答

### Q: 學生說看不到其他同學的作業？
**A:** 檢查觀摩模式設定：
- 如果是「完全關閉」→ 這是正常的
- 如果是「手動審核」→ 需要您審核通過才會顯示
- 如果是「自動公開」→ 檢查是否有作業被手動拒絕

### Q: 如何批量審核多個作業？
**A:** 目前需要逐一審核，建議：
1. 使用篩選功能按教室查看
2. 先評分再審核觀摩狀態
3. 定期處理避免積壓

### Q: 學生忘記教室代碼怎麼辦？
**A:** 在教室管理頁面：
1. 找到對應教室
2. 點擊「複製代碼」
3. 或直接提供完整連結給學生

### Q: 可以修改學生的作業內容嗎？
**A:** 可以！在作業管理頁面：
1. 點擊作業的「編輯」按鈕
2. 修改姓名、組別、標題、網址等
3. 還可以重新分配教室

### Q: 如何刪除不當的作業？
**A:** 在作業管理頁面：
1. 找到對應作業
2. 點擊「刪除」按鈕
3. 確認刪除（無法復原）

## 🔒 安全注意事項

1. **定期修改密碼**：建議每學期更換一次管理員密碼
2. **登出習慣**：使用完畢請點擊「登出」
3. **權限保護**：不要將管理員帳號密碼告訴學生
4. **備份重要資料**：重要的作業資訊建議截圖保存
